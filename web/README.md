# TrustCapture Web UI

React + TypeScript + Vite web application for TrustCapture photo verification platform.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```bash
cp .env.example .env
```

3. Start development server:
```bash
npm run dev
```

The app will be available at http://localhost:3000

## Backend API

Make sure the backend API is running at http://localhost:8000

## Features

- Authentication (Login/Register)
- Dashboard
- Campaign Management
- Vendor Management
- Photo Gallery
- Reports & Analytics
- Map Visualization

## Tech Stack

- React 18
- TypeScript
- Vite
- TailwindCSS
- React Router
- React Query
- Axios
- Plotly.js (Charts)
- Mapbox GL JS (Maps)
